# ChrisCandle

A fun and secure way to organize Secret Santa gift exchanges with friends, family, or coworkers. This app automatically matches participants, keeps assignments hidden, and sends out notifications so everyone knows who they're buying forâ€”without spoiling the surprise.

## Project Structure

```
src/
  components/     # Reusable React components
    ui/          # Basic UI components (buttons, inputs, etc.)
    features/    # Feature-specific components
  pages/         # Page components/routes
  lib/           # Utility functions and helpers
  hooks/         # Custom React hooks
  types/         # TypeScript type definitions
  styles/        # Global styles and themes
public/          # Static assets
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and fill in your environment variables

3. Run the development server:
   ```bash
   npm run dev
   ```
